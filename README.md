# single-page

> Single-page starter project, include webpack, less, babel.

## Dependencies

* Zepto
* Webpack
* Less
* Babel

## Usage

Make sure installed `workplus-cli`.

```bash
$ npm install workplus-cli -g
```

And run:

```bash
$ workplus start single-page my-project
```

## Build Setup

```bash
# install dependencies
npm install

# dev
npm run dev

# build production
npm run build

```
